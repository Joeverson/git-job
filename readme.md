# git-job

The script for easy create and update new branches with base the main branch.

# requisitos minimos 
- git
- install jq
- curl

## install


copy 'git-job' the script in your directory '/bin', above open the terminal(shell) and
call for ```$ git-job```, done!

## commands

For more informations

```
$ git-job --help
```
